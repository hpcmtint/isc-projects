The list contains client classes configured on the monitored DHCP servers. Check one or more classes you want to assign to the host reservation and click the Insert button. The list can be opened several times. The already selected classes are checked and cannot be unchecked in this list. To remove a class from a host reservation, remove its tag from the input box.

If a desired class is not in this list, you can type its name directly in the input box.

If the list is long, use the filtering box to find desired classes. Click on the filter button to select the search mode.
